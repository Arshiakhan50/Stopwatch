# Stopwatch
# Controls

`␣`: Start/stop

`r`: Reset

`y`: Copy time to clipboard

`e`: Toggle editing mode

In editing mode:

`j` or `←`: Move left

`l` or `→`: Move right

`0-9`: Replace highlighted digit with pressed digit

`h`: Move to hours

`m`: Move to minutes

`s`: Move to seconds

`d`: Move to deciseconds
